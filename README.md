# Pomodojo MVP

A clean, distraction-free focus timer and learning community prototype.

## How to Run

Simply open `index.html` in your web browser. No server or installation required.

## Features

- **Focus Timer**: 25m work / 5m break cycles. Works even in background tabs.
- **Session Tracking**: Counts your focus sessions (stored in browser).
- **Classrooms & Store**: UI mockups for future features.

## Tech Stack

- Vanilla HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (ES6+)
- Font: Outfit (Google Fonts)

Enjoy your focus session!
